# Student Grade Management System

## Project Overview

The Student Grade Management System is a Java-based console application developed as part of the CodeAlpha Java Programming Internship. The project is designed to manage student grades efficiently by allowing users to enter student information, calculate performance statistics, and generate summary reports.

This application demonstrates the use of Java programming fundamentals, including Object-Oriented Programming (OOP), ArrayLists, loops, conditional statements, and user input handling.

---

## Features

* Add student names and marks.
* Store student records using ArrayList.
* Calculate average marks.
* Find the highest score.
* Find the lowest score.
* Display a summary report of all students.
* Console-based interactive menu system.

---

## Technologies Used

* Object-Oriented Programming (OOP)
